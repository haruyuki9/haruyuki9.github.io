二叉树

性质

性质1 在二叉树的第 i 层至多有 2^(i －1)个结点。(i>=1)
性质2 深度为 k 的二叉树至多有 2^(k-1)个结点(k >=1)。
性质3 对任何一棵二叉树T, 如果其叶结点数为n0, 度为2的结点数为 n2,则n0＝n2＋1。
性质4 具有 n (n>=0) 个结点的完全二叉树的深度为【log2(n)】（下取整）＋1
性质5 如将一棵有n个结点的完全二叉树自顶向下，同层自左向右连续为结点编号0,1, …, n-1，则有：
1）若i = 0, 则 i 无双亲, 若i > 0, 则 i 的双亲为」(i -1)/2」

   2）若2*i+1 < n, 则i 的左子女为 2*i+1，若2*i+2 < n, 则 i 的右子女为2*i+2

   3）若结点编号i为偶数，且i != 0,则左兄弟结点i-1.

   4）若结点编号i为奇数，且i != n-1,则右兄弟结点为i+1.

   5）结点i 所在层次为」log2(i+1) 」

树和森林的相互转换

树转换成二叉树的画法

(1) 在兄弟结点之间加一条线
(2) 对每个结点，只保留它与第一个孩子的连线，抹去与其他孩子的连线
(3) 以树根为轴心，顺时针旋转45°
由树转换为二叉树，无右子树

森林转换成二叉树的画法

(1) 将森林中的每棵树转换成相应的二叉树
(2) 每棵树的根也可视为兄弟结点，在每棵树之间加一根连线
(3) 以第一棵树的根为轴心顺时针旋转45°
合并后右子树结点为除第一颗树外的其余树结点和

二叉树转换成树的画法

(1) 对于右子树，删除所有右侧分支
(2) 对于左子树，删除右侧分支。连接树根与被分离结点

排序

  1. 直接插入排序：比较第一第二位大小关系/依次将数放入temp，temp数值和前面素有元素比较，每比较失败一次后移，直到temp大于比较的元素
  2. 冒泡排序：每一次将最大的放到左边
  3. 快速排序：以第一位作为枢纽元，依次移动左右指针插空，左指针插小于枢纽元元素，右指针插大于枢纽元元素
  4. 归并排序，两个两个比较，比较完成四个四比较，以此类推，直到比较完成
  5. 堆排序：按照元素个数画出二叉树，每次遇到带孩子结点就和左右孩子比较，将大数放置到结点上，小数放置到孩子上
  6. 希尔排序：选择一个步长，按照步长进行分组比较
